# Gym Management System

A Java Full Stack web application developed for managing gym memberships, events, payments, and user activities efficiently.

## Features

* User Registration & Login
* Admin Dashboard
* User Dashboard
* Membership Management
* Recharge Plan Management
* Event Management
* Payment Tracking
* Profile Management
* CRUD Operations

## Modules

### Admin Module

* Manage members
* Manage gym events
* Monitor payments
* Update membership plans

### User Module

* User registration and login
* View membership details
* Recharge membership plans
* Manage profile information
* Access event details

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Java
* JSP
* Servlets

### Database

* MySQL
